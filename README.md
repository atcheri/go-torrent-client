## Go clean arch template (an attempt)

### Overview

A small project-template in go that aims to use clean-architecture

###

### Install dependencies

```shell
$ go mod tidy
```

### Run the app cli mode
```shell
$ go run ./cmd/main.go
```

### Run the app http mode
```shell
$ go run ./http/main.go
```

### Run the tests
```shell
$ go test ./...
```
